The program's main class is located in `./src/main/java/com/example/Main.java`. It is recommended to use Java 21 or higher.

Cucumber feature files are located in `./src/test/java/com/example/`, and the entry point is the `RunCucumberTest` class. Whitebox/JUnit tests are in `./src/test/java/com/example/WhiteBox/`

To run the cucumber tests in eclipse, right click on "RunCucumberTest.java", hover over "Run As" and click "JUnit Test".
To run the white-box tests in eclipse, right click on `./src/test/java/com/example/WhiteBox/`, hover over "Run As" and click "JUnit Test".
